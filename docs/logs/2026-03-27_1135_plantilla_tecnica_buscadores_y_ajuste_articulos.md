# [Docs/UI] - Plantilla tecnica de buscadores CRUD y ajuste visual en articulos

## Que se hizo
- Se creo una mini plantilla tecnica reutilizable para implementar buscadores CRUD con sugerencias en futuros modulos.
- Se ajusto el bloque de busqueda de `Articulos` para que quede alineado a la derecha siguiendo el patron visual actual.

## Por que
- Hacia falta una referencia corta y operativa para replicar el patron sin releer toda la guia extensa.
- El toolbar de `Articulos` habia quedado visualmente corrido respecto del lenguaje comun del backoffice.

## Impacto
- La mula ya tiene una receta corta para replicar buscadores en modulos nuevos, incluido CRM.
- `Articulos` queda mas consistente con el resto de los listados del sistema.

## Decisiones tomadas
- Se mantuvo la guia amplia como documento rector y se agrego una plantilla breve como apoyo de implementacion.
- El ajuste visual en `Articulos` fue minimo y sin tocar logica del modulo.
