# [UI] - Auth premium ligero para suite RXN

## Que se hizo
- Se mejoro visualmente el circuito auth con un fondo corporativo suave, card translucida y encabezados de marca mas cuidados.
- Se unifico el mismo tratamiento visual en login, recuperar acceso, reset y reenvio.

## Por que
- El flujo ya estaba corregido y branded, pero todavia podia verse mas premium sin dejar de ser liviano ni salirse del stack actual.

## Impacto
- La experiencia de acceso se siente mas corporativa y consistente con la suite RXN.
- Se mantiene responsive, server-rendered y sin dependencias extra.

## Decisiones tomadas
- Se aplico una mejora visual conservadora basada solo en CSS reutilizable y ajustes menores de markup.
