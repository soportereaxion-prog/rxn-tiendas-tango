# [UI] - Branding Reaxion completo en auth

## Que se hizo
- Se replico el branding de Reaxion en todas las vistas del flujo de autenticacion: login, recuperacion, reseteo y reenvio de verificacion.
- Se centralizo el estilo del logo auth en la capa global para mantener consistencia.

## Por que
- El login ya habia quedado alineado con la suite RXN y faltaba completar el mismo lenguaje visual en el resto de pantallas de acceso.

## Impacto
- Todo el circuito auth ahora se percibe como una sola experiencia de marca.
- Se mantiene responsive y consistente en mobile y desktop.

## Decisiones tomadas
- Se reutilizo el mismo logo institucional y el mismo bloque de sitio/correo para evitar variantes innecesarias entre vistas.
