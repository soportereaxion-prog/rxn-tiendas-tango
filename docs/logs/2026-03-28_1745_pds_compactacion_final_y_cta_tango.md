# [CRM/PDS] - Compactacion final y CTA visible de Tango

## Que se hizo
- Se mantuvo visible el boton `Enviar a Tango` tambien en alta, dejandolo deshabilitado hasta que el PDS exista guardado.
- Se redujeron gaps, paddings y alturas visuales del formulario PDS para sacar aire innecesario entre cajas.
- Se achicaron las textareas tecnicas lo justo para subir el contenido util sin perder legibilidad.

## Por que
- La pantalla todavia se sentia demasiado abierta para uso operativo diario.
- El usuario necesitaba ver desde el principio el flujo completo `Guardar` + `Enviar a Tango`, aunque el segundo paso dependa de guardar primero.

## Impacto
- El formulario queda mas compacto en escritorio y comunica mejor el circuito operativo real.
